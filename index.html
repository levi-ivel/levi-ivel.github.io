MostUsed

MostUsed is a plugin I made for note taking webapp Obsidian.

It's programmed in TypeScript and takes the 100 most used words from all the users notes and makes a graph of them using Chart.JS. It also updates in real time.

This project was started on 01-05-2024 and was merged with Obsidians branch on 18-05-2024

Data Size Converter

Data Size Converter is a simple application that takes 2 file sizes and a given value and converts that value to the new size type. It also has the option to turn on dark mode.

It's programmed in Flutter using Android Studio and was part of a pratical exam for school.

This project was started on 19-04-2024 and finished on 23-04-2024.

Desstop

Desstop is an ASCII faced desktop assistant that can preform numerous commands via a terminal. Dess can set countdowns, open programs, spin a wheel of set outcomes and much more.

It's programmed in Python using Tkinter and is a side project of mine.

This project was started on 19-05-2024 and the first release was published on 23-05-2024, but it's still being worked on.
