MostUsed is a plugin I made for note taking webapp Obsidian.


It's programmed in TypeScript and takes the 100 most used words from all the users notes and makes a graph of them using Chart.JS. It also updates in real time.


This project was started on 01-05-2024, was merged with Obsidians branch on 18-05-2024 and is currently sitting at 120 downloads

Desstop is an ASCII faced desktop assistant that can perform numerous commands via a terminal. Dess can set countdowns, open programs, spin a wheel of set outcomes and much more.


It's programmed in Python using Tkinter and is a side project of mine.


This project was started on 19-05-2024 and is still being worked on.
