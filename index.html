MostUsed is een plugin die ik heb gemaakt voor notitie-app Obsidian.


Het is geprogrammeerd in TypeScript en neemt de 100 meest gebruikte woorden van een gebruiker en zet deze in een grafiek met behulp van Chart.JS. Deze grafiek update in realtime.


Dit project was begonnen op 01-05-2024, was met Obsidians branch samengevoegd op 18-05-2024 en zit nu op 180 downloads.

MostUsed is a plugin I made for note taking webapp Obsidian.


It's programmed in TypeScript and takes the 100 most used words from all the users notes and makes a graph of them using Chart.JS. This graph also updates in real time.


This project was started on 01-05-2024, was merged with Obsidians branch on 18-05-2024 and is currently sitting at 180 downloads.

Desstop is een ASCII-gezichts desktop assistent dat meerdere opdrachten kan uitvoeren via een terminal.


Het is geprogrammeerd in Python met gebruik van Tkinter en het is een zijproject van mij. Dess kan timers zetten, programma's openen, een wiel draaien dat een willekeurig resultaat laat zien en nog veel meer.


Dit project was begonnen op 19-05-2024 en er word nogsteeds aan gewerkt.

Desstop is an ASCII faced desktop assistant that can perform numerous commands via a terminal.


It's programmed in Python using Tkinter and is a side project of mine. Dess can set countdowns, open programs, spin a wheel of set outcomes and much more


This project was started on 19-05-2024 and is still being worked on.

MAD is een bestands-converter dat met maximaal 15 decimalen accuraat bestand groottes kan converteren tussen elkaar.


Het is geprogrammeerd in Flutter met gebruik van Android Studio en was mijn keuzedeel voor mobile app development. Het heeft een home, dark mode en calculatie pagina.


Dit project was begonnen op 19-04-2024 en was afgemaakt op 23-04-2024.

MAD is a simple application that takes 2 file sizes and a given value and converts that value to the new size type.


It's programmed in Flutter using Android Studio and was part of a pratical exam mobile app development for school. It contains a home, dark mode and calculations page and can convert files within 15 decimals


This project was started on 19-04-2024 and finished on 23-04-2024.
